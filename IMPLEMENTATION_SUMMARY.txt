TECHNICAL STACK
================================================================================

Frontend:

├─ Language: TypeScript 5.2
├─ Styling: CSS3 (inline + global)
├─ Routing: Angular Router (new standalone API)

└─ Build Tool: Angular CLI + esbuild

Architecture:
├─ Component-based (no NgModules)
├─ Standalone components throughout
├─ Feature-based folder structure
├─ Two-way data binding with FormsModule
└─ Responsive design-ready

================================================================================
QUICK START
================================================================================

1. Add your company logo:
   - File: frontend/src/assets/logo.png
   - Format: PNG with transparency
   - Size: 200x200px minimum
   - See LOGO_SETUP.md for details

2. Install dependencies:
   cd frontend
   npm install

3. Run development server:
   npm start
   → Opens at http://localhost:4200/

4. Test features:
   - Home page: View news feed
   - Auth page: Try login/signup
   - Profile page: Check user info
   - Navigation: Use header buttons
